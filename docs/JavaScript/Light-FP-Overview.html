# Monoid

wike

Monoid 可理解为具有单位元的半群(semigroup)。定义如下：

假设 S 是一个集合(Set)，存在二元运算 *，S * S -> S，则满足：

  1. Associativity

集合 S 内的元素 a, b 和 c，有 (a * b) * c = a * (b * c)

  1. Identity element

集合 S 内存在元素 e，对于集合内的任何元素 a，有 e * a = a * e = a
