# 范畴学基础理论

近似的说法，**范畴学是一门针对函数的(抽象)代数理论的分支**。范畴学起源就是 *对象之间的函数系统*<sup>1</sup>。

## 术语

Category - 范畴
Sets - 集合

## 范畴的定义

在定义之前，先描述集合(Set)间的函数关系。按照 G. Cantor 的想法，由一些确定的且相互区别的对象(Object)汇集而组成的一个整体，称为集合。组成该集合的对象称为它的元素<sup>2</sup>，描述为 `a ∈ A`，其中 `a` 是集合 `A` 中的元素。
