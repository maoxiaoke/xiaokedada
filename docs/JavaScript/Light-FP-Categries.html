# 集合的函数关系

在定义之前，先描述集合(Set)间的函数关系。按照 G. Cantor 的想法，由一些确定的且相互区别的对象(Object)汇集而组成的一个整体，称为集合。组成该集合的对象称为它的元素2，描述为 𝑎 ∈ A，其中 𝑎 是集合 A 中的元素。

集合间的关系可以描述为：

𝑓 : A → B

𝑓 表示集合 A 到集合 B 的函数。典型的集合和函数的关系如下：

上图表达的一个这样的函数关系，即

若 𝑓 : A → B

若 𝑔 : B → C

则存在 𝑔∘𝑓 : A → C

这就称为 函数组合(composite)。满足以下关系：

(𝑔∘𝑓)(𝑎) = 𝑔(𝑓(𝑎))    𝑎 ∈ A.

Let's Go A Litte Farther! 若还有函数关系：ℎ : C → D。

那么：

(ℎ∘𝑔)∘𝑓 = ℎ∘(𝑔∘𝑓)

代数学中，即为 结合率。

最后，对于任意的集合 A，都有一个单位函数(Identity Function)：

1A: A → A

单位函数可以作为“组合”操作的“单位元”(unit)。即：

𝑓∘1A = 𝑓 = 1B∘𝑓

# 范畴

从数据的角度，范畴由以下数据组成：

  • Objects: A, B, C,...
  • Arrows: 𝑓, 𝑔, ℎ,...

满足：

  • 对于每个 arrow 𝑓，有两个给定的 objects

dom(𝑓), cod(𝑓)

称为 𝑓 的域(domain) 和值域(codomain)。对于:

𝑓 : A → B

则，A = dom(𝑓) 和 B = cod(𝑓)。

  • 给定 arrows 𝑓 : A → B 和 𝑔 : B → C，即

cod(𝑓) = dom(𝑔)

则有：

𝑔∘𝑓 : A → C

称为 𝑓 和 𝑔 的组合(composite)。

  • 对于任意 A，有

1A: A → A

称为 A 的 identity arrow。

  • 满足结合率(Associativity)

(ℎ∘𝑔)∘𝑓 = ℎ∘(𝑔∘𝑓)

  • Unit:

𝑓∘1A = 𝑓 = 1B∘𝑓

满足上述所有定义，便是范畴。值得强调的一点是，Objects 不必非是集合；Arrows 不必非是函数。如果你熟悉群论(Groups)，范畴可以认为是一般化的群。

有时，arrows 也被称为 morphisms(态射)。
